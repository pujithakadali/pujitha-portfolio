# Cyber Threat Intelligence Portfolio

A modern Cyber Threat Intelligence portfolio showcasing cybersecurity projects, AI-driven solutions, and secure web development skills using Python, Django, and modern frontend technologies.

## Features

* Cybersecurity-focused portfolio design
* Threat Intelligence themed interface
* Project showcase section
* Skills and tools display
* Responsive modern UI
* Contact and social links

## Tech Stack

| Category       | Technologies                                   |
| -------------- | ---------------------------------------------- |
| Framework      | TanStack Start (Full-Stack React with SSR/SSG) |
| Frontend       | React 19 + TypeScript                          |
| Build Tool     | Vite 7                                         |
| Styling        | Tailwind CSS v4                                |
| UI Components  | shadcn/ui + Radix UI                           |
| Icons          | Lucide React                                   |
| Routing        | TanStack Router                                |
| Data Fetching  | TanStack Query (React Query)                   |
| Forms          | React Hook Form + Zod Validation               |
| Server Runtime | Cloudflare Workers                             |
| Utilities      | clsx, tailwind-merge                           |
| Code Quality   | ESLint, Prettier                               |


## Purpose

This portfolio was created to showcase my interest and skills in:

* Cyber Threat Intelligence
* Secure Web Development
* AI-based Security Solutions
